# 6502 Emulator

## Overview

The 6502 Emulator is a software project aimed at simulating the functioning of the 6502 microprocessor. This emulator allows users to execute 6502 assembly language instructions and observe the results as if running on actual hardware.

## Features
- **Complete 6502 Instruction Set**: All instructions supported by the 6502 processor.
- **Memory Management**: Simulates the memory architecture of the 6502, allowing for read and write operations.
- **Debugging Tools**: Provides tools for stepping through instructions, examining memory, and monitoring CPU registers.

## Installation

To install the 6502 Emulator, clone this repository:

```bash
git clone https://github.com/theddaywork/6502-emulator.git
```

Navigate to the project directory:

```bash
cd 6502-emulator
```

## Usage

Once installed, you can run the emulator using the command line interface. Load your assembly files and start the emulation.
